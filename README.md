# Next Auth

```bash
# create application
$ npx create-next-app@Latest auth-tutorial

# initialize shadcn
$ npx shadcn-ui@latest init

# other required deps
$ yarn add -D prisma
$ yarn add @prisma/client

```

### Prisma

```bash
# initialize prisma
$ npx prisma init

# optional free postresdb on neon.tech
# generate prisma (NEED INTERNET CONNECTION)
$ npx prisma generate

# push changes to db
$ npx prisma db push

# open prisma studion on browser
$ npx prisma studio
```

### AuthJS

```bash
# goto authjs.dev
# install dependencies
$ yarn add @auth/prisma-adapter

# refer the docts here https://authjs.dev/guides/upgrade-to-v5
```
